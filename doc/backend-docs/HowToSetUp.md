# How to Set Up & Host the Backend Server

## Prerequisites

| Requirement | Version | Windows | macOS |
|---|---|---|---|
| **Node.js** | 20+ | [nodejs.org](https://nodejs.org) or `winget install OpenJS.NodeJS.LTS` | `brew install node@20` |
| **npm** | 10+ (ships with Node) | Included with Node.js | Included with Node.js |

## macOS

1. Install Node.js (if not Installed)
   Open terminal and run `brew install node@20` (install using HomeBrew)

2. Go to the Server Folder in terminal
   `cd /LilaAssignment_PavanChavan/Server`

3. run `npm install` to install all the Dependancies

4. run `npm run dev` to start the Server

5. This will starts the server at `http://localhost:5000`.

## Windows
1. Install Node.js (if not Installed)
   open command prompt as Administrator
   run `winget install OpenJS.NodeJS.LTS`

2. Go to Client (Game) build Folder
   `cd /LilaAssignment_PavanChavan/Server`

3. run `npm install` to install all the Dependancies

4. run `npm run dev` to start the Server

5. This will starts the server at `http://localhost:5000`.
